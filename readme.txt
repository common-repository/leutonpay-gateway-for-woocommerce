The light and secure payment plugin for WooCommerce in Cameroon accepting MTN Mobile Money, VISA and MasterCard

== Description ==

The LeutonPay WooCommerce Integration plugin enables seamless integration of the LeutonPay payment gateway with your WooCommerce-powered online store. LeutonPay offers a secure and reliable payment processing solution, allowing you to accept payments from various sources conveniently. 


== Third-Party Integration ==

This plugin relies on the third-party service provided by LeutonConnect. LeutonConnect acts as a secure intermediary for payment transactions, enabling users to make purchases using various payment methods, including MTN Mobile Money, VISA, and MasterCard...


== Supported Payment Methods ==

LeutonConnect supports multiple payment methods, including:
   MTN Mobile Money: Customers can conveniently pay for their orders using their MTN Mobile Money accounts.


== How It Works ==

When a customer proceeds to checkout on your WooCommerce store, the LeutonPay integration handles the payment processing. The plugin securely transmits the necessary payment details to the LeutonConnect API endpoint (https://proxy.leutonconnect.com), where the payment transaction is processed.


== Why LeutonPay? ==

LeutonPay offers several benefits for WooCommerce merchants:
   - Security: LeutonPay ensures the security of payment transactions through robust encryption and compliance with industry standards.
   - Convenience: With LeutonPay, customers can make payments using various methods, including credit/debit cards, digital wallets, and bank transfers.
   - Customization: The plugin provides options for customizing the checkout experience to align with your branding and preferences.


== Setting Up the Integration ==

To enable payment processing via LeutonConnect, you'll need to sign up for an account with LeutonConnect and obtain the necessary API credentials. Once you have the credentials, you can configure the LeutonPay WooCommerce Integration plugin settings to establish the integration with LeutonConnect, you'll need to:
   - Install and activate the plugin from the WordPress Plugin Directory.
   - Sign up for a LeutonConnect account at LeutonConnect (https://portal.leutonconnect.com).
   - Obtain API credentials (API key and secret) from your LeutonConnect account dashboard.
   - Configure the plugin settings with your LeutonConnect API credentials to establish the integration.
